## Getting Started
main.js
```javascript
...
process.env['NODE_ENV'] = 'development';
```

shell
```shell
npm run start
```

## Build

main.js
```javascript
...
process.env['NODE_ENV'] = 'production';
```

shell
```shell
npm run build
```
